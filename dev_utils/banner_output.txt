TOHDO 📒 is a terminal based interactive to-do list.                                                
                                                                                                    
Tohdo let's you add, delete, filter, show tasks and let's you                                       
edit status of each tasks. It also let's you add tags.                                              
                                                                                                    
TOHDO stand on the shoulders of giants 👼                                                           
- Typer for command line tool     https://github.com/tiangolo/typer                                 
- Rich for the beautiful terminal https://github.com/Textualize/rich
